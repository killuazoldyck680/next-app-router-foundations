A professional, full-stack administrative dashboard built with the Next.js App Router. This project demonstrates a modern web development workflow, from database management to secure authentication and optimized UI patterns.

## 🛠️ Tech Stack & Tools

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Database:** [Neon](https://neon.tech/) (Serverless Postgres)
* **Authentication:** [NextAuth.js](https://next-auth.js.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Validation:** [Zod](https://zod.dev/)
* **Icons:** [Heroicons](https://heroicons.com/)
* **Deployment:** [Vercel](https://vercel.com/)

## ✨ Key Features

-   **Secure Authentication:** Protected routes and session management using NextAuth.js and Middleware.
-   **Dynamic Dashboard:** Real-time data visualization for total collected, pending, and total invoices.
-   **Full CRUD Functionality:** Create, Read, Update, and Delete invoices with server-side validation.
-   **Optimized Performance:** Implementation of **Streaming** and **Suspense** to handle slow data fetches gracefully.
-   **Advanced Search & Pagination:** Seamlessly filter through invoices using URL search parameters for a shared state.
